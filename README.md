#MHFS
Minimal HTTP File Server

First written in 2006, updated and published to GitHub in 2016.

#Capabilities

* Rate limiting
* Choose specific files to serve
* Choose a directory to serve
* Supports uploading

#Goals

* Jar file distribution
* Simple to use
* Clean up the 10 year old code

#Usage
```
$ java -jar MHFS.jar
```
